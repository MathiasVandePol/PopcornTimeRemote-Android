Control Popcorn Time (popcorntime.io) v0.3.3 and higher directly from your phone.

Choose any movie or series, add it to your bookmarks or start playing it. Play, pause or seek through the video, all from the comfort of your couch.

Although the application supports most of the versions of Popcorn Time, make sure you're running the latest version to use the latest features.

Currently, this application is in beta and every suggestion is welcome! Issues can be reported on GitHub and translations are always great.
